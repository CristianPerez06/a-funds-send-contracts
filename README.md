# a-funds-send-contracts

Contracts to be used along with a-funds-send-dapp.

## Tech stack

- Typescript
- Prettier, ESLint, and Husky
- Hardhat

## How to execute the app

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```
